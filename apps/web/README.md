# EdenFinance

## Visão Geral

Aplicação web para gestão financeira pessoal: transações, metas, investimentos, relatórios e dashboards. Stack: Vite, React, TypeScript, Tailwind CSS e shadcn-ui.

## Desenvolvimento

```sh
npm i
npm run dev
```

## Build

```sh
npm run build
```

## Preview de build

```sh
npm run preview
```

## Tecnologias

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

## Deploy

Hospede o conteúdo de `dist/` em qualquer provedor estático (Netlify, Vercel, Cloudflare Pages, GitHub Pages ou servidor próprio). Configure SPA fallback para `index.html`.
